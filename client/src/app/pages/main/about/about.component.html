About the Mod
Momentum Mod is a free, standalone game built on the Source Engine that aims to faithfully recreate, centralize, and improve upon the movement game modes found in Counter-Strike, Team Fortress, Half-Life 2, and Titanfall.
We aim to provide the best experience found from all of your favorite servers, all in one convenient place, all for free.
But it gets better!
With access to the native code of the Source Engine itself, favorite features from plugins and completely original features can be natively implemented and included with the base game. This means absolutely no extra requirements or dependencies needed; no server/plugin setup tutorials need be followed; it's all here when you click "Install Game"!
Being a standalone game means we can optimize for the supported game modes, giving Momentum Mod a smoother, native feel and the subsequent performance of being a game made for them, compared to traditional multiplayer servers with network latency and hefty, high-level plugin code running all on top of the game itself.
We are actively taking feedback from the community at large to develop a game that feels perfect, runs smoothly, and most importantly, is fun to play!
Some of Momentum Mod's features:
  • Single-player client - no multiplayer game server connection required!
  • Client-side auto-hop - no lag when jumping!
  • Surf ramp bug fixes - no more random ramp bugs!
  • Slope jump fixes - consistent speed gained from jumping down slopes!
  • Original entities - easier mapping and interesting new opportunities for game modes!
  • Replays and analytics - for every run submitted!
  • Global leaderboards - for every map submitted!
  • Integrated map selector - easy browsing and one-click downloads for maps, all from in-game!
  • Hassle-free multiplayer - one-click multiplayer with Steam Lobbies (with community-focused dedicated servers coming later!)
