About the Game
Momentum Mod is a free, standalone game built on the Strata Source Engine that aims to faithfully recreate, centralize, and improve upon the movement game modes found in Counter-Strike, Team Fortress, Half-Life 2, Titanfall 2, and Quake. We aim to provide the best experience found from all of your favorite games and servers, all in one convenient place, all for free.
But it gets better!
With access to the native code of the Source Engine itself, favorite features from plugins and completely original features can be natively implemented and included with the base game. This means absolutely no extra requirements or dependencies needed; no server/plugin setup tutorials need be followed; it's all here when you click "Install Game"!
Being a standalone game means we can optimize for the supported game modes, giving Momentum Mod a smoother, native feel and the subsequent performance of being a game made for them, compared to traditional multiplayer servers with network latency and hefty, high-level plugin code running all on top of the game itself.
This is a game being built by Source movement fans, for Source movement fans! We are actively taking feedback from the community at large to develop a game that feels perfect, runs smoothly, and most importantly, is fun to play!
Some of Momentum Mod's confirmed features:
  • Single-player client - no multiplayer game server connection required to play!
  • Fully licensed Source Engine project - full compatibility with CS:GO/Portal 2 and TF2/CS:S maps, Portals, Panorama UI, and more!
  • Client movement fixes - smooth gameplay with auto-jump, no ramp bugs, and consistent slope jump speeds built right into the client!
  • Original entities - easier mapping and interesting new opportunities for all supported game modes!
  • Global leaderboards, replays, and analytics - for every run you do, on every map submitted!
  • Integrated map selector - easy map browsing and one-click downloads for maps, all from in-game!
  • Hassle-free multiplayer - one-click multiplayer with Steam Lobbies, including cross-map chat!
  • Community-driven development - public GitHub issue tracker, monthly Q&A sessions, direct feedback in Discord!
Some of Momentum Mod's planned features:
  • In-game tutorials - no more watching random YouTube videos to learn; learn through playing!
  • Fully custom campaigns - designed to introduce complete newbies to the modes, story and speedrun modes included!
  • Community-inspired dedicated servers - an experience many players are used to!
  • Community-focused customization - a game-wide cosmetic shop with items unlocked purely through gameplay!
