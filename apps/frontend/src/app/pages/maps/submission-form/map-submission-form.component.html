Drag selected images around to change their order, and make sure the image in the thumbnail slot is your favorite, as it will be used very frequently in the UI.

Credits

Momentum profiles for all contributors to the map. For breakdown of credit types, see the docs.

If you know their Steam account but not their name in Momentum, you can search for their Steam Community ID, which can be found with a site like steamid.io. Anyone who has ever played Momentum or signed in to our website should have a Momentum account. If they haven't, you can submit a "placeholder" credit that will generate a placeholder user for the given alias when the map is approved.

Leaderboards

Momentum leaderboards are relative to a specific gamemode, and maps can have multiple leaderboards in different modes.

When a map enters submission we generate temporary leaderboards for (almost) all gamemodes. Players can submit runs and leave reviews recommending official leaderboards on any of them, even if you don't suggest them here.

For detailed info see the docs.

Private Testing

Private testing is an optional first phase of map submission where only players you personally invite can play and review your map. Each player needs to explicitly Accept your invitation. (This can, for example, be done through the Notifications menu.)

This is intended to make sharing and versioning your map files simpler. Rather than share each change manually over Discord or wherever, you can just submit new versions via this site and their game clients will download the latest version when launching.

If you want to start in Private Testing, select the checkbox below. You can keep the submission in Private Testing for as long as you want provided you're still actively working on the map. Once you're done, you can move the map forward to the Content Approval stage. If you don't care about Private Testing, leave the box unchecked and the submission will go straight to Content Approval.
